# B2B 콘텐츠 주제 리서치 Skill

## 언제 사용하나요
사용자가 이번 달 B2B 클라우드/IaaS 콘텐츠 주제와 발행 일정을 만들고 싶을 때 사용합니다.

## 필요한 입력
- 대상 월
- 서비스/브랜드 정보
- 현재 캠페인 또는 강조할 메시지
- 참고할 고객 질문, 키워드, 기존 콘텐츠가 있으면 함께 사용

## 참고 자료
- 고객/제약: `references/audience-and-rules.md`
- 리서치 절차: `references/research-workflow.md`
- 출력 형식: `references/output-template.md`

## 작업 절차
1. 대상 월의 평일과 공휴일을 확인합니다.
2. `https://library.gabia.com/`에서 클라우드/IaaS 콘텐츠 패턴을 참고합니다.
3. SMB C레벨과 개발 팀장 페르소나의 질문을 뽑습니다.
4. 구매 여정 1, 2, 3단계별 후보 주제를 8~12개 만듭니다.
5. 고객 문제, 검색 의도, 브랜드 적합성 기준으로 4~5개를 고릅니다.
6. 최종 확정 전 담당 서비스 마케터 확인이 필요하다고 표시합니다.
7. 주 1회, 주말/공휴일 제외 기준으로 발행 일정을 짭니다.

## 완료 전 확인
- 월 4~5건 이내입니다.
- 주 1회 발행입니다.
- 주말과 공휴일을 피했습니다.
- 구매 여정 1~3단계가 반영됐습니다.
- 경쟁사명은 직접 쓰지 않았습니다.
- 보안/장애 관련 직접 표현을 피했습니다.
- 조회수 또는 검색 순위에 기여할 이유가 있습니다.
