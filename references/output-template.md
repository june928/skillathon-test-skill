# 출력 형식

## 후보 주제
| 우선순위 | 구매 여정 | 페르소나 | 주제 | 선정 이유 | 주의 표현 |
|---|---|---|---|---|---|

## 발행 일정
| 주차 | 발행일 | 구매 여정 | 페르소나 | 콘텐츠 주제 | 형식 | 고객 질문 | 검색 의도 | 핵심 메시지 | CTA |
|---|---|---|---|---|---|---|---|---|---|

## 완료 전 확인
- 4~5건인지 확인
- 주말/공휴일 제외 확인
- 구매 여정 매핑 확인
- 경쟁사명 직접 언급 없음
- 보안/장애 직접 표현 없음
- 마케터 확인 필요 여부 표시

## CTA 예시
- 1단계: 상담 신청, 견적 문의, 체크리스트 다운로드
- 2단계: 무료 상담, 프로세스 가이드, 사례 콘텐츠
- 3단계: 뉴스레터 구독, 가이드 다운로드, 관련 콘텐츠 읽기
